# ESM `delay`

This repository provides an implementation of a `delay` function compatible with
ESM.

## Installation & Usage

### Browser

```javascript
import delay from 'https://tomashubelbauer.github.io/esm-delay/index.js';

await delay(100);
```

### Node

Use[`timers/promisers`](https://nodejs.org/api/timers.html#timers-promises-api)
